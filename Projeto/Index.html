Projeto

Neste projeto constarão 7(sete) páginas web, sendo que 3(três) páginas são sobre o tema Reflorestamento, 3(três) sobre a Linguagem de Programação Perl e uma página Portal, que conecta as outras 6(seis) e apresenta dados importantes sobre o Projeto.

As páginas foram feitas a partir de estruturação em HTML e design em CSS, e podem ser acessadas pelas imagens abaixo, pelos links no cabeçalho desta página ou pelos links no rodapé desta página.

Conteúdos apresentados nas páginas:
  • Introduções sobre os assuntos
  • Exemplificações dos temas
  • Dados qualitativos e quantitativos
  • Formulário para arrecadação de dinheiro
  • Recomendações Bibliográficas
